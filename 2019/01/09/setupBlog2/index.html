配置文件

  1. 站点文件_config.yml，修改Extension字段，配置theme，设置为indigo
  2. 设置头像，在根目录/source/ 创建images/， 存放头像图片
  3. 配置主题文件_config.yml, 可以参考indigo configuration
  4. hexo g -d 查看远程效果
  5. hexo g, hexo s 查看本地效果
